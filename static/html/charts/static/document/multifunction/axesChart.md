
## 坐标系多功能图配置表
暂支持柱状图、点阵图、折线图和区域图，也可以任意配置这些图形中的一种和几种的组合图

```javascript
config = {
    /**
     * 图表主体配置
     */
    chartBody: {
      /**
       * 图表的宽（包含padding值）
       */
      width: 600,
      /**
       * 图表的高（包含padding值）
       */
      height: 400,
      /**
       * 图表的内边距
       */
      padding: {
        top: 40,
        right: 40,
        bottom: 40,
        left: 40
      }
    }
}
```
